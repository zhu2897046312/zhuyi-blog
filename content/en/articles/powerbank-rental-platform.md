## Project Overview

The Overseas Power Bank Rental Platform is an O2O platform providing shared power bank rental services, including user rental H5 pages and merchant backend management system. The project needs to support high-performance mobile experience and map positioning functionality.

![preview](/articles/launch-your-porfolio.jpg)

## Tech Stack

- **Vue3** - Progressive JavaScript Framework
- **Vite** - Next Generation Frontend Build Tool
- **Vant** - Lightweight and Reliable Mobile Vue Component Library
- **Tailwind CSS** - Atomic CSS Framework
- **Ant Design Vue** - Enterprise-level UI Component Library (Backend System)
- **Google Maps API** - Map Service API

## My Responsibilities

### 1. H5 User End Development

Responsible for the development of the rental platform H5 user end, using Vue3 + Vant + Vite tech stack, through componentization and code optimization, creating a high-performance and smooth mobile user experience.

### 2. Map Function Integration

Successfully integrated Google Maps API, implementing map positioning, search, and route navigation functions for rental outlets, providing key technical support for core business processes.

### 3. Backend Management System

Participated in backend management system development, using Ant Design Vue to complete interactive interface implementation for device management, order statistics, etc.

## Project Highlights

- ✅ High-performance mobile experience with smooth user interactions
- ✅ Complete map positioning and navigation functionality
- ✅ Device management and order statistics functionality
- ✅ Responsive design, adapting to various mobile devices

## Project Results

Successfully delivered a fully functional O2O rental platform, providing users with convenient power bank rental services while offering efficient device management tools for merchants.
