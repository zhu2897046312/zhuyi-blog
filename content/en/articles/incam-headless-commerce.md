## Project Overview

The incam Company Website & E-commerce Platform is a website platform combining company website and independent e-commerce attributes, using headless architecture to achieve content management and frontend rendering, supporting SEO optimization and global access. The project uses Shopify as the CMS backend management system and fetches data through Storefront API.

![preview](/articles/launch-your-porfolio.jpg)

## Tech Stack

- **Nuxt.js** - Server-Side Rendering Framework
- **TypeScript** - Type-Safe JavaScript Superset
- **Tailwind CSS** - Atomic CSS Framework
- **Shopify Storefront API** - GraphQL API
- **Webflow** - Web Design Tool
- **Figma** - UI Design Tool
- **Vercel** - Deployment Platform

## My Responsibilities

### 1. Project Architecture Setup

Responsible for overall project architecture setup, using Shopify as the CMS backend management system, fetching product, collection, order, and other data through Storefront API (GraphQL), achieving data-driven frontend rendering.

### 2. Frontend Page Development

Based on Nuxt.js + TypeScript framework, combined with Tailwind CSS to build responsive frontend pages, including homepage, product list, detail pages, and backend management modules, ensuring code type safety and efficient development.

### 3. UI Design and Deployment

Assisted in using Webflow and Figma for UI design and prototype iteration, optimizing user interaction experience, and finally deployed through Vercel platform, achieving automatic CI/CD and global CDN acceleration.

## Project Highlights

- ✅ Headless e-commerce architecture with flexible content management

Successfully delivered a modern headless e-commerce platform, providing the company with flexible content management and efficient e-commerce functionality, while achieving global fast access and automatic deployment through the Vercel platform.
