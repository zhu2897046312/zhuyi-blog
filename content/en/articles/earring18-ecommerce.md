## Project Overview

The Cross-border Jewelry E-commerce Platform (earring18.com) is a global B2C jewelry e-commerce platform that needs to balance both PC and mobile user experience with SEO optimization. The project uses modern frontend tech stack and improves first-screen loading speed and SEO performance through server-side rendering.

![preview](/articles/launch-your-porfolio.jpg)

## Tech Stack

- **Vue.js** - Progressive JavaScript Framework
- **Nuxt.js (SSR)** - Server-Side Rendering Framework
- **Naive UI** - Vue 3 Component Library
- **Tailwind CSS** - Atomic CSS Framework
- **Element Plus** - Vue 3 Component Library (Backend System)

## My Responsibilities

### 1. Frontend Interface Development

Based on the Nuxt.js framework, responsible for responsive interface development and core interaction logic implementation for both PC and mobile platforms, effectively utilizing server-side rendering to improve first-screen loading speed and SEO performance.

### 2. Component Development

Using NaiveUI and TailwindCSS to build consistent design language and highly reusable components, ensuring unified visual experience and development efficiency for users.

### 3. Backend Management System

Independently completed partial module development of the e-commerce backend ERP system using Vue + Element Plus, supporting management of products, orders, users, and other information.

## Project Highlights

- ✅ Dual-platform responsive design, perfect adaptation for PC and mobile devices
- ✅ Server-side rendering optimization, improving SEO and first-screen loading speed
- ✅ Unified component library and design language
- ✅ Complete backend management system

## Project Results

Successfully delivered a fully functional, high-performance cross-border e-commerce platform, providing users with a smooth shopping experience while offering convenient backend management functions for merchants.
